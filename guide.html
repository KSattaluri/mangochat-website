About Mango Chat

Totally free, open-source, lightweight, AI-powered voice productivity app for Windows.

Written in Rust with extremely low memory usage, Mango Chat skips local models like Whisper, Chromium-based transcription, and Windows native speech recognition. Instead, it streams your audio directly to your chosen cloud speech-to-text providers for ultra-low-latency results.

Providers like Deepgram and AssemblyAI offer generous free credits—$250 or up to 700+ hours—with No credit card required. Get started at zero cost and accelerate your productivity right away. NOW!

20–40 MB typical memory usage—a fraction of Electron-based alternatives

Installation & Getting Started

  1. Download MangoChat-Setup.exe from the latest GitHub release
  2. Run the installer — no admin privileges needed
  3. Accept defaults or change the install path
  4. Click Install, then Finish
  5. Search Mango Chat in the Windows search bar to launch the app

Settings are stored in %LOCALAPPDATA%\MangoChat\. Uninstalling preserves your settings and keys.

Configuration

  1. Open Settings (gear icon)
  2. Go to Providers
  3. Select a provider (Deepgram, AssemblyAI, OpenAI, or ElevenLabs)
  4. Paste your API key
  5. Click Verify to test the connection
  6. Click Save — you're done

API keys are encrypted with Windows DPAPI. Never stored in plain text.

Personalization

  • Appearance — change color theme, window opacity, anchor position
  • Display — follow cursor across monitors or pin to one
  • Commands — add browser commands (e.g. "open github" opens a URL)
  • Programs — add programs to launch by voice (e.g. "launch claude")
  • Aliases — create text shortcuts (e.g. "my email" expands to your address)
  • VAD — switch between Strict and Lenient voice activity detection
